# Einbuergerungstest Plugin Changelog

## [Unreleased]

## [1.3.0] - 2026-03-30

### Added

- A checkbox to disable the random question order
- A progress bar to show the progress of the quiz
- An action to reset quiz statistics

### Fixed

- Do not restart the quiz on the tool window hide

## [1.2.0] - 2026-02-15

### Added

- Show question images
- Keyboard navigation
- Show the number of questions in each category
- Show question statistics

### Changed

- Do not show the same question twice

## [1.1.0] - 2026-01-28

### Added

- General question category
- Mark a question as favorite

## [1.0.0] - 2026-01-25

### Added

- Initial release
